用户：是指注册、登录、使用本平台软件，能够享受本平台提供的各类功能服务的个人、企业或其他组织（以下尊称“您”）。其中，用户在选择新建工单时，为需求方用户（委托商）功能角色，本平台将按照需求为其匹配可提供相应服务的用户；经本平台审核能够提供相应设备维护、IT服务及其衍生服务为服务方用户功能角色。用户根据自身需求及能力，经本平台审核后可切换角色，享受相应的平台功能。

新建工单：指需求方用户在畅修平台上发布需求工单，简称发布工单。

接单：指有相应维护能力且能够提供相应服务的用户在本平台上接收工单并提供相应维护服务。

四、服务流程及注意事项

1、新建工单时

（1）用户新建工单时，应按照平台提示，真实完整地填写保修工单内容，所提供的联系方式须真实有效，如因联系方式错误造成接单匹配失败的，由发布错误信息的用户自行承担相应损失。

（2）用户在新建工单时可上传设备故障图片或其他相关附件，但在上传及发布时应注意自身设备的信息安全，如因用户自身或其他非本平台原因造成的信息泄密，本平台不承担相关责任。

2、服务进行中

（1）用户在平台新建工单需求后，平台会根据工单需求自动匹配并提单给具备相应资质的服务方用户，或根据用户的指令将工单提单给指定的服务方用户。

（2）具备相应资质的服务方用户接单后，需求方用户可选择远程处理或者现场服务，服务时间由双方约定。选择现场服务的，服务方用户不可擅自变更已确定的服务时间，须在约定的时间内完成现场签到。选择远程处理方式的，由服务方用户通过远程指导协助解决问题。

（3）服务方用户须保障自身服务质量，严格遵守平台制定的服务规则，提供服务时须参考行业标准及相应业务操作规则。

（4）在需求方用户新建工单信息发布后，尚未有服务方用户接单的，需求方用户可随时对工单内容进行修改或撤回。

（5）需求方发布的工单已被服务方用户接单但尚未实施服务前，经用户双方协商一致的，任何一方均可对工单发布、接单行为进行修改或撤回。

（6）需求方用户认为已接单的服务方用户不能满足其要求的，可向平台提出变更或转单申请。

（7）需求方用户发布指定的工单给服务方用户，但服务方用户无法提供服务的，可向平台提出转单申请，但须经需求方用户同意。

（8）用户可在自己的账号中对自己的工单进行查询及管理，企业用户可根据使用需求设置相应的企业用户管理人，平台对企业用户的管理人设置了不同角色的划分，不同角色的管理人享有不同的管理权限。

3、服务完成

服务完成后，由服务方用户所服务的最终客户签发《服务验收单》，对服务内容及结果进行当场确认；最终客户签字确认后，即视为本次服务内容结束，服务方用户仅对本次服务内容承担相应的质量保证，对超出本次服务范畴之外或无证据表明是服务方用户所导致的其他品质瑕疵、纠纷、损害均与服务方用户无关，需求方用户及其最终客户应自行承担相关责任。需求方用户应依据《服务验收单》与服务方用户进行服务费用结算。在使用本平台服务时，如因用户填报虚假信息或者使用不正当手段促成工单，导致用户财产损失或违法相关法律规定的，本平台不承担任何责任，由违约方向未违约的一方承担相关法律责任。
